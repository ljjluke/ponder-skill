### Step 1: 八卦镜发散 ⭐ 核心阶段

**⛔ 必须执行全部 5 轮发散。每轮对用户可见。不可合并。不可跳过。**

**5轮发散流程：**

```
Round 1: 基础审视 — F1→F8 逐卦深度分析

  ⛔ 每卦必须独立输出以下完整结构，不可一行带过：
  ┌─────────────────────────────────────────────┐
  │ F? ☰/☷/☳/☴/☵/☲/☶/☱ [卦名] — [当前任务的具体维度名]  │
  │                                             │
  │ 体(Ti): [通用本质 — 这个卦代表什么]            │
  │ 用(Yong): [当前任务的具体表现 — 在这里体现为什么] │
  │                                             │
  │ 子镜① [诸子A]: [具体推理 — 至少2-3句]          │
  │   → 洞察: [这个视角揭示的关键发现]              │
  │                                             │
  │ 子镜② [诸子B]: [具体推理 — 至少2-3句]          │
  │   → 洞察: [这个视角揭示的关键发现]              │
  │                                             │
  │ 文化类比: [跳出领域框架的类比 — 至少1个具体例子] │
  │   → 启示: [类比带来的新思路]                   │
  │                                             │
  │ 已知信息: [已掌握的3-5条关键信息]               │
  │ 盲区: [2-3个不确定/缺失的信息]                  │
  │ 初步想法: [基于以上分析产生的2-4个构想]          │
  │                                             │
  │ 评分: [0-10] — [打分理由一句话]               │
  └─────────────────────────────────────────────┘

  各卦子镜分配:
    F1 ☰ 乾(力量之源): 兵家(势从何来?敌友高地在哪?) + 縱横家(利益能否对齐?谁能争取?)
    F2 ☷ 坤(根基承载): 農家(土壤适合什么?季节对吗?) + 水利家(资源流向?上下游关系?)
    F3 ☳ 震(变动突破): 醫家(症状还是根因?有未病吗?) + 陰陽家(对立面是谁?转化点在哪?)
    F4 ☴ 巽(渗透传播): 工匠(核心工具够吗?庖丁解牛的关节间隙在哪?) + 禪家(剥离装饰后剩下什么?)
    F5 ☵ 坎(风险深渊): 史家(历史上谁犯过类似错误?结果?) + 道家(不干预是否更好?干预会制造新风险吗?)
    F6 ☲ 离(显眼依附): 工匠(什么支撑表面光鲜?隐藏结构?) + 儒家(谁被忽视了?人的需求被遮蔽了吗?)
    F7 ☶ 艮(边界止步): 法家(规则过时了吗?谁在执法?违规代价?) + 道家(知止是保护还是限制?)
    F8 ☱ 兑(汇聚共赢): 儒家(道德基线?谁的利益被牺牲?) + 縱横家(利益能重新分配吗?谁是天然盟友?)

Round 2: 交叉关联 — 关键维度对
  必须输出至少3对关联分析，每对包含:
  ┌─────────────────────────────────────────────┐
  │ 关联对: F? ↔ F? (分差=|?-?|=X)              │
  │ 卦象交互: 64卦对照 (hexagram-lookup)          │
  │ 理(Li): [通用模式 — 这对揭示的跨领域规律]      │
  │ 事(Shi): [当前具体表现 — 这对在当前任务中的含义] │
  │ 协同/冲突: [这对如何互相增强或互相制约]          │
  └─────────────────────────────────────────────┘

Round 3: 变化条件分析
  对Round 2的每对关联，区分稳定vs变化因素:
  ┌─────────────────────────────────────────────┐
  │ F?↔F? 变化分析:                              │
  │ 稳定因素: [1-2个不变量]                       │
  │ 变化因素: [1-2个正在变化的量]                  │
  │ 二阶效应: [如果变化因素位移→会发生什么连锁反应?] │
  └─────────────────────────────────────────────┘

Round 4: 盲区补全
  对分数<7的每个卦象，逐卦补全:
  ┌─────────────────────────────────────────────┐
  │ F? [?/10]: [盲区描述]                        │
  │ ①知识图谱: [查到什么/没查到]                   │
  │ ②WebSearch: [搜索发现] (≤3分卦象必须搜索)     │
  │ ③追问用户: [具体问题] (前面都解决不了才问)     │
  │ 补全后评分: [?→?/10]                         │
  └─────────────────────────────────────────────┘

Round 5: 发散自检
  5个问题逐一回答:
  ① 有没有卦象没有产生任何想法? → [是/否, 具体说明]
  ② 有没有想法太早被否决了? → [回顾被否决的想法, 是否有过早排除的]
  ③ 有没有未声明的假设? → [列出至少3条隐含假设]
  ④ 8卦是否拼成完整图景? → [缺失的拼图块是什么]
  ⑤ 有没有跳出领域框架? → [至少2个跨界联想的具体例子]
```
