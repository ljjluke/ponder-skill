## ☯️ 八面审视镜 — 道家八卦的决策智慧

源自道家八卦 (Bagua) 思想，8 个通用视角审视每一个决策：

| 卦象 | 审视面 | 核心问题 |
|------|--------|---------|
| ☰ 乾 | 力量之源 | 做这件事的驱动力从哪来？ |
| ☷ 坤 | 根基承载 | 承载这件事的底盘是什么？ |
| ☳ 震 | 变动突破 | 哪里可能发生意想不到的变化？ |
| ☴ 巽 | 渗透传播 | 怎么让效果真正渗透扩散开来？ |
| ☵ 坎 | 风险深渊 | 最深的坑在哪？最坏会怎样？ |
| ☲ 离 | 显眼依附 | 最引人注目的表面底下依附什么？ |
| ☶ 艮 | 边界止步 | 有什么绝对不能碰的线？ |
| ☱ 兑 | 汇聚共赢 | 各方利益怎么平衡？有没有都满意的方案？ |

> **抽象框架永远不变，具体维度由你的需求决定。**

## 🧠 MMA 经络记忆系统

源自《黄帝内经》经络学说。知识沿12条经脉如气血般流动，每个穴位是一条知识。

| 人脑能力 | MMA 引擎实现 |
|---------|------------|
| 看到问题 → 想起经验 | 得气 — 上下文与穴位共振，最佳匹配自然浮现 |
| 记不清了 → 努力回想 | 循经感传 — 刺激一个穴位，感觉沿经脉扩散 |
| 情绪强烈 → 记忆深刻 | 情绪调制器 — 恐+15, 怒+10, 喜+8 巩固分加成 |
| 遗忘≠删除，是提取失败 | 隐穴机制 — 标记隐藏而非删除，可被唤醒 |
| 工作记忆 7±2 组块 | 三焦气化 — 上焦7个(刚用过), 中焦(本会话), 下焦(历史) |
| 睡眠巩固记忆 | 睡眠回放 — 会话结束时加速回放+情绪加权巩固 |
| 专家组块化 | 腧穴集群 — 共现频繁的穴位自动形成知识组块 |
| 旧知识过时 → 纠偏 | 状态机 — 假设→待验证→已确认→争议→证伪→休眠→归档 |
| **🆕 情景 vs 语义记忆** | 情景记忆(经历)衰减快60d / 语义记忆(知识)衰减慢90d |
| **🆕 记忆再巩固** | 得气召回后30分钟不稳定窗口，窗口内可塑性×1.5 |
| **🆕 源监控** | 7级来源可靠性(亲历1.0→传闻0.2)，可靠源加权×1.1 |
| **🆕 精细加工** | 4级加工深度(浅/中/深/最深)→初始巩固分+0~+6 |
| **🆕 八纲辨证** | 表里寒热虚实→脉象(浮沉数迟缓滑涩平)→权重调整 |
| **🆕 四象成熟度** | 老阳/少阴/少阳/老阴 — 知识健康度四象限诊断 |
| **🆕 经脉分片存储** | 16经脉独立文件，单分片损坏只丢1/16，自动备份恢复 |

## 🌐 语言自适应

用户用什么语言提问，就用什么语言回答。内部英文引擎执行，输出自动翻译。Unicode 书写的脚本检测覆盖所有语言——中文、日文、韩文、阿拉伯文、俄文、印地文……任何语言都支持。

> 输出语言检测由 `language_guard.js` 保障。

## 📊 核心算法

**MCTS（UCB 选择 + Welford 方差更新）+ TD(λ)（时序差分 + 资格迹）。MMA（经络记忆算法）— 12经脉+奇经八脉，得气召回。**

```
用户消息
    │
    ▼
┌─────────────┐    ┌──────────────────┐    ┌─────────────────┐
│  发散引擎    │───▶│  推演引擎         │───▶│  仲裁引擎        │
└─────────────┘    └────────┬─────────┘    └─────────────────┘
                            │                        │
          ┌─────────────────┴────────────────────────┘
          ▼
┌─────────────────────────────────────────────────────────┐
│              🧠 Memory Agent（静默旁观子Agent）            │
│                                                          │
│  ① PRE_ENGINE ─── 得气召回 ────→ 注入上下文               │
│  ② DURING_DIVERGE ─ 感知七情 ─→ 情绪时间线                │
│  ③ POST_SIMULATE ── 阿是穴插入 ─→ 经脉穴位               │
│  ④ PRE_CONVERGE ── 阴阳对冲检测 → 有矛盾时谏言            │
│  ⑤ POST_EXECUTION ─ 补泻更新 ─→ TD闭环                   │
│  ⑥ SESSION_END ─── 睡眠回放 ──→ 记忆巩固                  │
│                                                          │
│  MMA 引擎: 12经脉 + 奇经八脉                               │
│  得气 · 子午流注 · 循经感传 · 补泻 · 阿是穴 · 隐穴        │
└─────────────────────────────────────────────────────────┘
```

```bash

## 🔄 升级

```bash
/plugin marketplace update luke     # 自动使用已注册的源
/plugin install luke                # 已安装则自动更新
/reload-plugins
```

> GitHub 网络不稳定时，用 Gitee Pages 源重新注册：`/plugin marketplace add https://luke2438107466.gitee.io/mcts-skill`，再执行升级。
```
