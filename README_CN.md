## ☯️ 八面审视镜 — 道家八卦的决策智慧

源自道家八卦 (Bagua) 思想，8 个通用视角审视每一个决策：

| 卦象 | 审视面 | 核心问题 |
|------|--------|---------|
| ☰ 乾 | 力量之源 | 做这件事的驱动力从哪来？ |
| ☷ 坤 | 根基承载 | 承载这件事的底盘是什么？ |
| ☳ 震 | 变动突破 | 哪里可能发生意想不到的变化？ |
| ☴ 巽 | 渗透传播 | 怎么让效果真正渗透扩散开来？ |
| ☵ 坎 | 风险深渊 | 最深的坑在哪？最坏会怎样？ |
| ☲ 离 | 显眼依附 | 最引人注目的表面底下依附什么？ |
| ☶ 艮 | 边界止步 | 有什么绝对不能碰的线？ |
| ☱ 兑 | 汇聚共赢 | 各方利益怎么平衡？有没有都满意的方案？ |

> **抽象框架永远不变，具体维度由你的需求决定。**

## 🧠 模拟人脑记忆

| 人脑能力 | 引擎如何实现 |
|---------|------------|
| 看到问题 → 想起经验 | 联想回忆 — 最相关记忆自动浮现 |
| 记不清了 → 努力回想 | 碎片补全 — 顺着线索拼凑完整画面 |
| 还是想不起 → 查资料 | 外部求证 — 查文档、搜代码、问用户 |
| 旧经验可能过时 → 纠偏 | 状态机 — 假设→验证→确认→争议→证伪→回滚 |
| 很久不用 → 慢慢遗忘 | 记忆衰减 — 30 天不使用自动归档 |
| 有人提起 → 突然想起 | 回忆触发 — 相关线索把归档知识拉回来 |

## 🌐 语言自适应

用户用什么语言提问，就用什么语言回答。内部英文引擎执行，输出自动翻译。Unicode 书写的脚本检测覆盖所有语言——中文、日文、韩文、阿拉伯文、俄文、印地文……任何语言都支持。

> 输出语言检测由 `language_guard.js` 保障。

## 🔧 三引擎决策流程

```
用户需求 → 约束收集
    │
    ▼
┌───────────────────────────────┐
│  发散引擎（头脑风暴）          │
│  ☯ 八面审视镜多次审视          │
│  🔍 六路侦查（代码/文档/竞品） │
│  🎡 视角轮盘（4~8 个视角）     │
│  ✂️  P0~P5 取舍（代码强制）    │
│                               │
│  → 2~8 个结构化的可行方案      │
└───────────┬───────────────────┘
            │
            ▼
┌───────────────────────────────┐
│  推演引擎（MCTS 树搜索）       │
│  🌲 Selection: UCB+K_bonus    │
│  🌿 Expansion: 展开新分支     │
│  🎲 Simulation: 快速推演到底  │
│  📈 Backpropagation: Welford  │
│                               │
│  → n / V / σ² / 置信度        │
└───────────┬───────────────────┘
            │
            ▼
┌───────────────────────────────┐
│  仲裁引擎（汇总与决策）         │
│  🏆 CLT-UCB 排序              │
│  🔍 自检（找漏洞+反向思考）    │
│  🕵️ 盲区审计（视角覆盖检查）   │
│  💾 TD 更新 → 写回知识图谱    │
│                               │
│  → 决策报告 + 执行计划         │
└───────────────────────────────┘
```

## 📊 核心算法

**MCTS（UCB 选择 + Welford 方差更新）+ TD(λ)（时序差分 + 资格迹）。知识图谱状态机（假设→验证→确认→争议→证伪→回滚）。**

```
用户消息
    │
    ▼
┌─────────────┐    ┌──────────────────┐    ┌─────────────────┐
│  发散引擎    │───▶│  推演引擎         │───▶│  仲裁引擎        │
│             │    │                  │    │                 │
│  八面审视   │    │  MCTS 树搜索     │    │  CLT-UCB 排序   │
│  跨视角关联 │    │  UCB 选择        │    │  自检            │
│  P0~P5 取舍 │    │  展开            │    │  盲区审计        │
│             │    │  模拟推演        │    │  TD 更新         │
└─────────────┘    │  回溯            │    └─────────────────┘
                   └────────┬─────────┘
                            │
                            ▼
                   ┌─────────────────┐
                   │  知识图谱        │
                   │                 │
                   │  状态机          │
                   │  假设→待验证→   │
                   │  已确认→争议→   │
                   │  证伪→休眠→     │
                   │  归档            │
                   └─────────────────┘
```

## 🚀 安装

```bash
# 国际 (GitHub)
/plugin marketplace add https://github.com/ljjluke/mcts-skill
/plugin install mcts-td-planner

# 国内 (Gitee)
/plugin marketplace add https://gitee.com/ljjluke/mcts-skill
/plugin install mcts-td-planner
```

装好后输入任意任务，看到 ⚡ 标志说明生效。

### ⚡ 记忆安全

知识图谱存储在 `~/.claude/data/skills/mcts-td-planner/`，与 skill 代码物理隔离。更新、重装都不会丢。想清除记忆删掉那个目录即可。
