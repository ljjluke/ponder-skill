## Phase 0.7: 神思 — 先审视前提, 再虚静, 后天游

**学术来源**: 《文心雕龙·神思》刘勰 + 《庄子·逍遥游》 + 认知偏误研究（Kahneman《思考，快与慢》）

### 核心框架

```
前提审视(检伪前提) → 虚静(清空) → 神凝(凝聚) → 神游(漫游) → 意象(浮现) → 言意(带回)
```

这不是"想几个疯狂点子"。这是先确认框架本身没歪，再让精神进入特定状态, 让东西自然浮出来。

**⛔ 在歪掉的前提上做神游，漫游得越远偏得越远。前提审视不可跳过。**

---

### 第零步: 前提审视 — Audit the Premises

**为什么要这一步**: 人在提需求时常带着思维陷阱——沉没成本、确认偏误、锚定、框架效应、可得性启发、幸存者偏差。这些陷阱藏在需求的"隐含前提"里。如果前提是错的，后面所有分析都在错的基础上做最优解。

**门控（E）— 先判断要不要审视**:
```
① 看问题赌注——用以下硬判据（不是凭"感觉"判，任一满足即高赌注）:
   - 不可逆: 决策做完无法低成本撤销（辞职、手术、签字、卖房）
   - 高成本: 错了损失大或难补救（投资、大额消费、方向选择）
   - 涉及他人: 影响别人且不可单方撤销（婚姻、人事、合作）
   - 长期影响: 后果持续半年以上（职业路径、定居、学习投入）
   ⛔ 判不清时按高赌注处理——宁可多做不可漏做（漏做=流程被掏空）。
   真正的可逆小事只有: 后果立刻可撤销且无成本（晚饭吃什么、取个变量名、
   文案措辞）。如果用户郑重其事来问"要不要X"，默认高赌注——
   会问就说明他在意，在意就不是可逆小事。
② 可逆小事（罕见，须同时满足:可逆+无成本+无他人影响+无长期后果）→ 跳过前提审视，直接进虚静。
③ 高赌注/复杂问题 → 进前提审视。赌注判定贯穿全管线——神思判高，
   后续发散互否/方案辩证/终态画像/归因/可谬/立场全部启用。
```

**前提可视化（D）— 零判定、零误报**:
```
① 列出本需求隐含的 3 个核心前提（"你的需求假设了X"）。
② 不说"你陷入了X偏误"——只把前提显式摆出来。
③ 常见陷阱对照（内部参考，不输出标签）:
   - 沉没成本: "已经投入了X，所以必须继续" → 前提是"过去的投入决定未来方向"
   - 确认偏误: "我倾向于找证据证明自己对" → 前提是"我的初始判断是对的"
   - 锚定: "被第一个数字/方案框住" → 前提是"第一个参照系合理"
   - 框架效应: "问题被表述方式限制" → 前提是"问题的表述方式等于问题本身"
   - 可得性启发: "最近看到的信息主导判断" → 前提是"容易想起来的=重要的"
   - 幸存者偏差: "只看成功案例" → 前提是"看得见的样本代表全部"
```

**用户参与式质疑（C）— 判定权交给用户**:
```
① 对列出的前提，用 AskUserQuestion 反问用户:"这个前提符合你的真实情况吗?"
② 用户否认的前提 → 标为伪前提，注入需求画像，后续步骤绕开。
③ 用户坚持的前提 → 标注"存疑"但不强行推翻（尊重用户判断，防误报）。
④ ⛔ 系统只提示不判定。误报由用户一句话否决，不伤信任。
```

**兜底机制**: 即使用户也在陷阱里（确认偏误导致自检失效），前提已被显式列出，后续八卦镜和辩论阶段还能二次发现。可视化本身就有暴露作用。

### 第一步: 虚静 — The Fasting of the Mind

**⛔ 不先虚静, 后面的所有发散都在旧框架内。跳过=假发散。**

刘勰: "陶钧文思, 贵在虚静, 疏瀹五藏, 澡雪精神"
庄子: "若一志, 无听之以耳而听之以心, 无听之以心而听之以气"

操作:
```
① 把当前问题放在一边。不看输入, 不想方案, 不考虑约束。
② 注意力转向呼吸或一个中性物(比如墙上的一个点、窗外的声音)。
③ 持续3-5个"呼吸周期"(在文本中就是3-5个自然的语句周期)。
④ 当感觉思维不再"用力抓取"时——进入下一步。

⛔ 虚静不是"放空大脑"——是"停止用力"。
```

### 第二步: 神凝 — Gather the Spirit

庄子说的"用志不分, 乃凝于神"——不分散注意力, 精神自然凝聚。

```
① 虚静之后, 不直接想问题。
② 只取问题中最核心的一个意象/一个词/一个矛盾点。
③ 盯着它, 不分析, 不想解决方案。
④ 让这个意象像涟漪一样自然扩散——它会自己带出关联。
```

### 第三步: 神游 — Spirited Wandering

**这是真正的"逍遥游"——不是6个固定视角, 是精神的无目的漫游。**

刘勰: "寂然凝虑, 思接千载; 悄焉动容, 视通万里"

```
① 从第二步的涟漪中, 让精神自然漫游:
   - 不要指定方向, 不预设"要看什么"
   - 一个联想带到另一个联想, 允许跳跃
   - 如果漫游到跟问题毫不相干的地方→那是好事
② 不评判:"这个关联有用吗?" 你现在不需要有用。
③ 持续漫游, 直到某个"有意思"的东西自动浮现。
```

### 第四步: 意象 — Images Emerge

庄子: "轮扁不能语斤" — 真正的精髓很难用语言说清楚, 但可以通过意象传达。

```
① 第三步中浮现的东西——它不一定是一个"方案"或"观点"。
② 它可能是一个画面、一种氛围、一个类比、一个印象。
③ 把它写下来, 不需要解释它"为什么"出现。
④ 现在, 看这个意象——如果它是一幅画, 画里有什么?
```

### 第五步: 言意 — Bring Back to the Problem

庄子: "得意而忘言" — 领会了意思就忘掉语言。反过来: 用语言把领会到的东西引出来。

```
① 从第四步的意象中, 找到和当前问题之间的连接点。
② 不需要牵强附会——如果有, 就是有。如果没有, 意象本身也丰富了你的视角。
③ 问自己:
   - 这个意象告诉了我什么之前没注意到的?
   - 如果这个意象是问题的答案, 答案是什么?
```
