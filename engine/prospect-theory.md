## 前景理论 — 损失态度 · 参照点 · 概率权重 (Prospect Theory Injection)

**学术来源**: Kahneman & Tversky (1979, 1992) — Prospect Theory + Cumulative Prospect Theory

**为什么要这一步**: 神思的前提审视已用 Kahneman 的认知偏差（确认偏误、锚定、可得性等），但偏差只是**认知操作的错误模式**。前景理论处理的是更底层的**价值感知结构**——人不只是在"想错"，人的价值感知本身就不是线性的。损失厌恶、参照点依赖、概率权重扭曲——这些不是偏差，是人的价值函数的固有形状。神思只触达了"怎么想错了"，没触达"怎么感知价值"。这一步补的是价值感知层。

### 核心机制

**三大结构性发现**（不是偏差清单，是价值函数的三个非线性）：

```
① 损失厌恶 (Loss Aversion): 损失的痛苦 ≈ 2.25倍 同等收益的快乐
   → 人不愿卖出亏损股票（损失实现=痛苦），过早卖出盈利股票（锁定收益=快乐）
   → 这不是"判断错误"——这是价值函数在亏损区比收益区更陡

② 参照点依赖 (Reference Dependence): 人不是评估绝对财富，而是评估相对于参照点的变化
   → 同一结果，参照点不同，感知价值完全不同
   → 参照点通常=现状/买入成本/社会比较/预期
   → 这不是"锚定偏差"——锚定是信息处理错误，参照点是价值感知的结构性特征

③ 概率权重扭曲 (Probability Weighting): 人对概率的感知是非线性的
   → 小概率被高估（对罕见风险的过度反应）
   → 中高概率被低估（"大概率没事"→忽视尾部风险）
   → 确定性效应：确定收益比大概率收益有溢价，确定损失比大概率损失更被回避
```

### 在神思中的注入点

注入位置：神思「前提可视化」段，在现有偏差对照表之后。

不是新增步骤——是在列出认知偏差后，多问三个"价值感知层"问题：

```
损失态度自检（不输出标签，只内部对照）:
- 当前判断中，是否因为"不想承认损失"而坚持某方向？（损失厌恶）
- 当前判断中，"现状"是否被当成了理所当然的参照点？（参照点依赖）
  → 如果参照点从"现状"换成"三个月前"或"目标状态"，判断会变吗？
- 当前判断中，是否高估了某个罕见但可怕的风险？（概率权重扭曲）
  → 如果这个风险的概率不是"有可能"而是具体数字（比如0.3%），反应会变吗？
```

⛔ 这三个问题不要求用户回答——它们是框架自己的内部自检。如果有发现，以自然语言融入神思的"前提审视发现"展示，不标注"前景理论"标签。

### 与现有范式的边界

- **vs 神思偏差检测**: 偏差是认知操作的错误模式（"我想错了"），前景理论是价值感知的结构性特征（"我的价值函数长这样"）。前者可纠正，后者需要被认知和补偿。
- **vs 可谬标注**: 可谬是"结论A最可能因X错"（判断层），前景理论是"我对价值的感知本身就偏了"（感知层）。前景理论不重复可谬，它是可谬的上游——感知偏了，基于感知的判断自然可谬。
- **vs 工作立场**: 立场有倾向+依据，前景理论让框架觉察到"这个倾向本身可能来自损失厌恶而非理性选择"。不改变立场机制，只增加一个自省维度。
